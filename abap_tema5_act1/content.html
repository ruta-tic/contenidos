1. ¿Qué es OPEN SQL?

OPEN SQL

OPEN SQL es un derivado SQL patentado que proviene de la compañía SAP. Esta última tiene la intención de proporcionar un dialecto SQL unificado para todas las bases de datos compatibles con SAP. OPEN SQL es parte del lenguaje de programación ABAP y también cuenta con una versión para Java.
